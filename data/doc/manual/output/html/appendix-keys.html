Appendix C - Shortcut Keys
  • Ctrl-b : Open the Dialogs dialog.
  • Ctrl-d : Delete a selected QSO.
  • Ctrl-e : Log export dialog.
  • Ctrl-f : Find a QSO in the log.
  • Ctrl-g : Activate the preferences dialog.
  • Ctrl-h : Open the manual.
  • Ctrl-i : Log import dialog.
  • Ctrl-j : Activate the scoring window.
  • Ctrl-k : Click on all buttons present and fill in all defaults.
  • Ctrl-l : Activate the defaults dialog
  • Ctrl-m : Merge logs.
  • Ctrl-n : Create a new log.
  • Ctrl-o : Open a log.
  • Ctrl-p : Print a log.
  • Ctrl-q : Exit program.
  • Ctrl-r : Activate the keyer.
  • Ctrl-s : Save the log.
  • Ctrl-t : Show/Hide the toolbar.
  • Ctrl-u : Update a selected QSO.
  • Ctrl-w : Write a qso to the log.
  • Ctrl-y : Clear the QSO frame.
  • Ctrl-z : Show a list of keys used by xlog.
  • Ctrl-1 : Show DXCC totals.
  • Ctrl-2 : Show WAC totals.
  • Ctrl-3 : Show WAS totals.
  • Ctrl-4 : Show WAZ totals.
  • Ctrl-5 : Show IOTA totals.
  • Ctrl-6 : Show Locator totals.
  • Ctrl-7 : Show Locator map.
  • Ctrl+space : Add a QSO and Click all buttons (Control w and Control k).
  • Control+Up Arrow : Move rig frequency 50 Hz up.
  • Control+Down Arrow : Move rig frequency 50 Hz down.
  • Control+Shift-b : Open/close the worked before window.
  • Control+Shift-c : Dupe check.
  • Control+Shift-d : Show contry list.
  • Control+Shift-e : Log editor.
  • Control+Shift-f : Find unknown countries.
  • Control+Shift-h : Activate the hamlib window.
  • Control+Shift-i : Info dialog.
  • Control+Shift-o : Sort log by date.
  • Control+Shift-p : Page Setup dialog.
  • Control+Shift-s : Save As.
  • Control+Shift-t : Show hamlib information.
  • Control+Shift-w : Close a log.
  • Control+Shift-x : Connect to the DX cluster.
  • Control-F10 : Access the menu.
  • PgUp : Scroll the log up.
  • PgDn : Scroll the log down.
  • Home : Go to the beginning of the log.
  • End : Go to the end of the log.
  • Up Arrow : Select previous QSO.
  • Down Arrow : Select next QSO.
  • Alt 1, Alt 2, etc : Switch between logs.
  • Alt-a : Select Awards entry.
  • Alt-c : Select Call entry.
  • Alt-d : Select Date entry.
  • Alt-e : Select End-utc entry.
  • Alt-h : Select QTH entry.
  • Alt-i : Select QSL-In box.
  • Alt-k : Select Remarks entry.
  • Alt-l : Select Locator entry.
  • Alt-m : Select Mode entry
  • Alt-n : Select Name entry.
  • Alt-o : Select QSL-Out box.
  • Alt-p : Select Power entry.
  • Alt-r : Select Rx-rst entry.
  • Alt-t : Select Tx-rst entry.
  • Alt-u : Select UTC entry.
  • Alt-z : Select mhZ entry
  • Alt-? : Lookup Callsign
