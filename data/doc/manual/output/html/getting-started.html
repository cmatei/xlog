Step 1 : Launch the Xlog Application

Launch the Xlog application by the following command: 

xlog <Log File Name - Optional>


Step 2 : Select to Create a New LogFile

Create a new log file by either the menu "Log" -> "New" or with the shortcut key CTRL-N


Step 3 : Log QSO's on the left pane

To enter the current date, simply click on the Date button


Step 4 : Click Write to save the QSO

Once all the fields are entered for the QSO click the Write button.

Many of the dialogs and configuration options are available via shortcut keys. See Appendix C - Shortcut Keys for a complete list.


Editing Log Entries

A log entry can be changed by selecting a row in the log, changing the QSO in the QSO frame on the left, and clicking Update.


Deleting Log Entries

QSO's can be deleted from the log by selecting a row in the log list and clicking Delete.


Clearing QSO Pane

Selecting "Edit" -> "Clear All" from the menu will clear all the fields in the QSO frame.
